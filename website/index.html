the pipeline

From API key to radar chart in one command.

01

Connect

Point the benchmark at any model API — Gemini, Claude, GPT, Qwen, or your own endpoint.

--provider gemini --model gemini-2.5-pro
02

Generate

Fresh challenges are created on-the-fly across 4+ domains. Every session is unique. No memorization possible.

--eval-challenges 20 --seed 42
03

Evaluate

Responses are scored on 5 dimensions: answer correctness, reasoning quality, confidence calibration, format compliance, and efficiency.

--scoring-profile balanced
04

Profile

An 8-axis radar profile is computed. Models are ranked on a multi-dimensional leaderboard. No single-number reductionism.

Results saved to results/EVAL_*.json

Core Capabilities

Reasoning

Multi-step math & probability

Tests ability to follow complex chains of thought without losing track of constraints. e.g., Markov chain probabilities.
Code Generation

Write working functions

Tests functional correctness and efficiency of generated code against execution environments and test suites.
Instruction Following

Precise formatting rules

Tests adherence to strict length, format, and content constraints (similar to IFEval).
Logical Deduction

Puzzles & constraints

Tests deductive reasoning using logic puzzles, ordering tasks, and constraint satisfaction problems.
Classification

Contextual sentiment

Tests basic NLU capabilities by categorizing text with varying degrees of built-in ambiguity.

Advanced Capabilities

Hallucination Resistance

Knowing what it doesn't know

Tests the model's ability to refuse unanswerable questions or identify false premises instead of confidently hallucinating.
Summarization

Factual compression

Tests ability to condense text while strictly preserving key numerical and entity facts.
Tool Use Planning

API sequence planning

Tests the model's ability to orchestrate multi-step action plans using a provided set of tools.
Information Extraction

Structured parsing

Tests ability to reliably extract entities and relationships from noisy text into structured JSON.
Anomaly Detection

Finding outliers

Tests pattern recognition and outlier detection in long numerical sequences.
